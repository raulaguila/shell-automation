# 📦 Node Version Manager (NVM)

The right way to install Node.js on Linux/macOS without permission issues.

## 📥 Installation

1.  **Run the install script:**
    ```bash
    curl -o- https://githubusercontent.com | bash
    ```

2.  **Restart your terminal** or run:
    ```bash
    source ~/.bashrc
    ```

## 🚀 Usage

*   **Install latest LTS version:** `nvm install --lts`
*   **Install specific version:** `nvm install 20`
*   **Switch versions:** `nvm use 18`
*   **List installed versions:** `nvm ls`

---
*Avoid using `sudo npm install`. NVM keeps packages in your user folder.*
